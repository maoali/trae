# 贪吃蛇游戏

一个使用HTML5 Canvas开发的经典贪吃蛇游戏。

## 功能特点

- 基于Canvas的流畅游戏画面
- 响应式设计，适配不同屏幕尺寸
- 实时分数显示
- 游戏速度随分数提升
- 简洁美观的用户界面
- 游戏结束提示和重新开始功能

## 技术栈

- HTML5 Canvas
- JavaScript
- CSS3

## 游戏规则

1. 使用键盘方向键控制蛇的移动方向
2. 吃到食物可以增加分数和蛇的长度
3. 撞到墙壁或自己的身体会导致游戏结束
4. 游戏速度会随着得分增加而提升

## 如何开始

1. 克隆或下载项目到本地
2. 使用浏览器打开`index.html`文件
3. 开始游戏！

## 游戏控制

- ↑: 向上移动
- ↓: 向下移动
- ←: 向左移动
- →: 向右移动
